# 🏥 Hospital Management System

A simple full-stack **Hospital Management System** built using **⚡ FastAPI** and **React.js**.

This project demonstrates how a **modern frontend (React)** can interact with a **high-performance backend built with FastAPI** to perform CRUD operations.

---

# 🚀 Tech Stack

### Backend
⚡ **FastAPI**  
🐍 Python  
📦 Uvicorn  

### Frontend
⚛️ React.js  
📡 Axios (API requests)  
🎨 CSS

# ⚡ Why FastAPI?

This project uses **FastAPI**, one of the **fastest Python web frameworks**.

FastAPI provides:

✔ High performance (comparable to NodeJS and Go)  
✔ Automatic API documentation (Swagger UI)  
✔ Easy request validation  
✔ Clean and modern Python code  
✔ Built-in async support
